is_distributed_normally

Richel Bilderbeek

2017-05-10

This vignette demonstrates the is_distributed_normally function.

library(wiritttea)

Here we create a normally distributed collection of values:

nd <- rnorm(n = 1000, mean = 0.0, sd = 1.0)
plot(nd)

As we know that they are normally distributed, is_distributed_normally should return true:

testit::assert(is_distributed_normally(nd))

Now we create a non-normal collection of values:

nnd <- runif(n = 1000, min = 0.0, max = 1.0)
plot(nnd)

As we know that they are not normally distributed, is_distributed_normally should return false:

testit::assert(!is_distributed_normally(nnd))

Use of dplyr

Here we’ll try to do something smart:

df_nd  <- data.frame(
  method = rep("nd", times = length(nd)), x = nd
)

df_nnd <- data.frame(
  method = rep("nnd", times = length(nnd)), x = nnd
)
df <- rbind(df_nd, df_nnd)
df$method <- as.factor(df$method)
