Demo of the summary statistics

Richel Bilderbeek

2017-05-10

There are multiple phylogeny summary statistics:

  • [Femke and Jolien summary statistic]
  • Gamma statistic
  • Colless statistic

First it is shown how a parameter file is built up, then the individual summary statistics are shown

Parameter file build-up

For this vignette, we need these libraries:

library(ape)
library(wiritttea)
options(warn = 2)

First we obtain the full path of a parameter file:

filename <- find_path("toy_example_1.RDa")
print(filename)
## [1] "/home/p230198/R/x86_64-pc-linux-gnu-library/3.4/wiritttea/extdata/toy_example_1.RDa"

This path will be different: Windows users will have a path starting with C:/ or D:/ or higher. GNU/Linux users will have a UNIX path.

The function read_file reads the file (it also checks if the file does indeed exist):

file <- wiritttes::read_file(filename)

file, which is a file, has a (hopefully) convenient structure. This structure can be observed using names:

print(names(file))
## [1] "parameters"    "pbd_output"    "species_trees" "alignments"   
## [5] "posteriors"

The (unsampled) (original) incipient species tree is in file$pbd_output, let’s see what’s there:

print(names(file$pbd_output))
##  [1] "tree"           "stree_random"   "stree_oldest"   "stree_youngest"
##  [5] "L"              "sL_random"      "sL_oldest"      "sL_youngest"   
##  [9] "igtree.extinct" "igtree.extant"  "recontree"      "reconL"        
## [13] "L0"

The (unsampled) (original) incipient species tree is in file$species_tree, let’s plot it:

incipient_species_tree <- file$pbd_output$igtree.extant
colors <- setNames(c("gray", "black"), c("i", "g"))
phytools::plotSimmap(
  incipient_species_tree, 
  colors = colors
)

I chose to plot the incipient species tree for extant species, thus igtree.extant. Because an incipient species trees has states on the edges (good and incipient), it is not a phylogeny of class phylo but has additional data as well.

Gamma statistic

The gamma statistic was introduced by Pybus and Harvey (2000).

  • gamma < 0: a tippy tree
  • gamma > 0: a stemmy tree

The incipient species tree looks like this:

colors <- stats::setNames(c("gray", "black"), c("i", "g"))
testit::assert(
  length(
    wiritttes::read_file(filename)$pbd_output$igtree.extant$tip.label
  ) > 0
)
phytools::plotSimmap(
  file$pbd_output$igtree.extant, 
  colors = colors
)

I will use a sampled species tree as my true tree:

true_tree <- wiritttes::get_species_tree_by_index(file = file, sti = 1)
plot(true_tree)

Note that [[1]][[1]]: it allows for multiple sampled species trees for more complex parameter files, but here it just looks ugly.

From that true tree, I measure the gamma statistic:

true_gamma <- ape::gammaStat(true_tree)
print(true_gamma)
## [1] 1.011407
if (true_gamma < 0) {
  print("tippy tree")
} else {
  print("stemmy tree")
}  
## [1] "stemmy tree"

The question is: how is the distribution of that statistic in the posterior?

This toy example has only one posterior file:

Extracting the trees from that posterior:

posterior_trees <- wiritttes::get_posteriors(file)[[1]][[1]]$trees
n_trees <- length(posterior_trees)
print(paste0("There are ", n_trees))
## [1] "There are 10"

For giggles, let’s plot all 10 posterior trees:

for (i in seq(1:n_trees)) {
  plot(posterior_trees[[i]])
}

From those posterior trees, we collect the gamma statistic:

gamma_statistics <- rep(0, times = n_trees)
for (i in seq(1:n_trees)) {
  gamma_statistics[i] <- ape::gammaStat(posterior_trees[[i]])
}
hist(gamma_statistics, xlim = c(-1, 1))
abline(v = true_gamma, col = "red")

It appears that all gamma statistics in the posterior are lower than the gamma statistic of the true tree.
